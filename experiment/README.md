## Structure
| Path    | Functionality |
| -------- | ------- |
| /src/ |   Contains the source code |
| /postprocessing/  | Contains the scripts that created the results for the MT incl. plots    |
| /preprocessing/    | Contains the scripts that created the fixed split for SEN12 and the HE/IHC presplits  |
| /results/ |   Contains the results of the src/full_estimator_pipeline.ipynb (All backbones with all estimators)  |
| /logs/ |   Holds the log files of the experiments (/src/experiments/) related to the thesis.  |
